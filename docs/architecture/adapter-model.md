# Adapter Model

Adapters are gameplay workflows built on top of Minecraft bindings.

Bindings translate Minecraft primitives, such as inventories, containers, items, and slots, into the contracts expected by platform-neutral Coffer authority modules. Adapters compose those bindings into player-facing workflows.

Player trade is future work and is not scaffolded yet. The inventory binding should be established first so later adapters can build on a stable, tested Minecraft inventory boundary.
