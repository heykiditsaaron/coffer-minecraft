# 0086 Experiential Contact Principles

## Summary

Record the first experiential-contact principles for future player/admin-facing
surfaces.

This is a principles note, not UI design, not gameplay design, and not a chat
format specification.

## Principles

- Prefer actionable refusal over non-actionable refusal.
- Prefer gentle guidance over verbose explanation when both are truthful.
- Prefer state visibility over narration.
- Prefer ledger or state surfaces over tutorial prose when the surface is about
  lawful reality.
- Preserve truthful omission instead of filling every gap with explanation.
- Use non-punitive wording where possible without hiding the actual condition.
- Preserve experimentation and discovery.
- Reduce futile retry loops by making non-actionable state visible when earned.
- Preserve player and admin reasoning ability.
- Avoid counterfeit reassurance.
- Avoid hostile or systemic tone.
- Earn explanation surfaces instead of making them universal.
- Keep trustworthy outcomes as the central experiential objective.

## Rationale

The repository now has stable enough accountability geometry to start naming the
experience principles that should govern future interaction surfaces.

The goal is not maximum friendliness or maximum permissiveness. The goal is a
sane, humane, and truthful interaction with lawful reality.

That means:

- preserving bounded-lawful-reality framing
- preserving accountability
- preserving reconstructability
- preserving historical honesty
- preserving anti-counterfeit behavior

## Scope

Included:

- the principles that should guide future interaction surfaces
- the emphasis on truthful omission and earned explanation
- the distinction between humane guidance and counterfeit reassurance

Excluded:

- UI implementation
- gameplay implementation
- chat layouts
- wording catalogs
- exact interface copy
- doctrine formalization
- final experiential authority claims

## Verification

No implementation changed for this note.

## Uncertainties

These are current principles, not permanent doctrine. Future contact surfaces
may require narrower wording or different presentation choices, but they should
still answer to the same accountability and truthfulness constraints.
